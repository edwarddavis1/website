LLM-Powered Chatbot Demo

This page demonstrates how to integrate a Hugging Face LLM-powered chatbot into your website. The chatbot appears in the bottom-right corner of the page.

How to Set Up the LLM Chatbot:

  1. Sign up for a free account at huggingface.co
  2. Go to https://huggingface.co/settings/tokens to create an API token
  3. Replace YOUR_HUGGING_FACE_API_TOKEN in the chatbot-llm.js file with your actual token
  4. Include the following lines in your HTML:
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/5.15.4/css/all.min.css">
<link rel="stylesheet" href="chatbot-llm.css">
<script src="chatbot-llm.js"></script>

Features of this Chatbot:

  • Connects to Hugging Face's hosted LLM models (like Llama-2)
  • Fallback to keyword-based responses if API is unavailable
  • Responsive design that works on mobile devices
  • Typing indicators for a more natural conversation feel
  • Automatically turns text references into clickable links

Note: The free tier of Hugging Face's Inference API has usage limits. For high-traffic websites, you may need to upgrade to a paid plan.
